`performance_campanhas.csv`: **dicionário de dados**

| Feature           | Descrição                                   |
|-------------------|---------------------------------------------|
| objetivo_campanha | objetivo da campanha de marketing           |
| visualizacoes     | total de visualizações obtidas nos anúncios |

|          Feature |                    Description |

|       pay_normal | number of times normal payment |
